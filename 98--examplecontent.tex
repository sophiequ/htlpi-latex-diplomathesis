\section{Additional Features}

You can use abbreviations using the \verb+\ac{}+ command like this: \ac{LTL}, 
where the abbreviation is expanded on first use and then printed normally: \ac{LTL}. 

Enumerations
\begin{enumerate}
	\item Foo
	\item Bar
\end{enumerate}

and itemizations
\begin{itemize}
	\item One
	\item Two
	\item Three
\end{itemize}
